# Student Exam Pass Prediction System

## Project Overview
This project predicts whether a student is likely to **Pass** or **Fail** an examination using **Logistic Regression**. The system analyzes student-related factors such as study hours, attendance, previous exam scores, parental education level, internet access, and extracurricular activities.

## Objectives
- Perform Exploratory Data Analysis (EDA)
- Identify factors affecting academic performance
- Predict Pass/Fail outcomes using Logistic Regression
- Classify students into risk categories
- Create visualizations and dashboards for decision-making

## Dataset
**Source:** Kaggle

**Dataset Name:** Student Performance Prediction Dataset

### Features
- Student_ID
- Gender
- Study_Hours_per_Week
- Attendance_Rate
- Past_Exam_Scores
- Parental_Education_Level
- Internet_Access_at_Home
- Extracurricular_Activities
- Final_Exam_Score

### Target Variable
- Pass_Fail

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Workflow
1. Data Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis
4. Feature Encoding
5. Logistic Regression Modeling
6. Model Evaluation
7. Risk Analysis
8. Dashboard Visualization

## Model Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix
- ROC-AUC Score

## Risk Categories
### High Risk
- Low attendance
- Low study hours
- Low past exam scores

### Medium Risk
- Average attendance and performance

### Low Risk
- High attendance
- Strong academic performance

## Visualizations
- Pass vs Fail Distribution
- Gender Distribution
- Study Hours Distribution
- Attendance Distribution
- Correlation Heatmap
- Scatter Plots
- Box Plots

## Expected Outcome
The system helps educational institutions identify at-risk students early and provide timely academic support to improve student success rates.
